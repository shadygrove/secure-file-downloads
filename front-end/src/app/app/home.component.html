Welcome Home

The goal of this project is to act as a jump-start for prototyping new components with this custom Angular Material theme palette.

A couple examples demoing usage of material components and how to use notifications have been added to the menu.

Get Started

To start prototyping, fork this StackBlitz project and rename it.

Under the Examples module, create a component to render your custom component.

Under the Shared folder, create your custom component.
